# Excel history clean

====

Overview

## Description

VB Script for clean history list of Microsoft Excel.

## Requirement

* Microsoft Windows
* Microsoft Excel

## Usage

Double click or run command line.

```
wscript ExcelHistotyClean.vbs
```

## Install

Copy ExcelHistotyClean.vbs your PC.
