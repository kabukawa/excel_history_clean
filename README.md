# Excel history clean

====

Overview

## Description

VB Script for clean history list of Microsoft Excel/Word.

## Requirement

* Microsoft Windows
* Microsoft Excel
* Microsoft Word

## Usage

Double click or run command line.

```
wscript ExcelHistotyClean.vbs
```

Or

```
wscript WordHistotyClean.vbs
```

## Install

Copy Script *.vbs your PC.
